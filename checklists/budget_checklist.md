# 經費編列檢核清單

> 確保經費編列符合 SBIR 規定，避免因經費問題被退件。

## 比例檢查
- [ ] 人事費 ≤ 60%
- [ ] 委託研究費 ≤ 30%
- [ ] 技術移轉費 ≤ 10%
- [ ] 總經費符合補助上限

## 人事費
- [ ] 主持人薪資合理
- [ ] 核心成員薪資合理
- [ ] 投入比例明確
- [ ] 有薪資證明
- [ ] 勞健保單獨列出

## 消耗性器材
- [ ] 項目明確
- [ ] 數量合理
- [ ] 單價合理
- [ ] 與計畫相關

## 設備使用費
- [ ] 使用租用而非購置
- [ ] 租金合理
- [ ] 使用期間明確
- [ ] 有報價單

## 委託研究費
- [ ] 委託項目明確
- [ ] 委託單位資格符合
- [ ] 費用合理
- [ ] 有合約或報價

## 差旅費
- [ ] 出差目的明確
- [ ] 次數合理
- [ ] 費用符合規定

## 其他費用
- [ ] 項目明確
- [ ] 金額合理
- [ ] 與計畫相關

## 對應性檢查
- [ ] 每筆支出都有對應工作項目
- [ ] 經費與時程對應
- [ ] 經費與人力對應

## 佐證文件
- [ ] 有報價單
- [ ] 有合約（如適用）
- [ ] 有薪資證明

## 成本效益
- [ ] 有成本效益分析
- [ ] 投資回報率合理
- [ ] 效益可量化
