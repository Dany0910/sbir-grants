# Phase 1 計畫書撰寫檢核清單

> Phase 1 簡報格式（12-15 頁），本檢核清單協助確保內容完整。

## 第 1 頁：封面
- [ ] 計畫名稱清楚
- [ ] 公司名稱
- [ ] 申請階段（Phase 1）
- [ ] 申請日期

## 第 2-3 頁：公司簡介
- [ ] 公司基本資料
- [ ] 核心業務
- [ ] 團隊優勢
- [ ] 過往實績

## 第 4-5 頁：問題陳述
- [ ] 目標客戶明確
- [ ] 痛點具體且量化
- [ ] 有產業數據佐證
- [ ] 有客戶訪談證據
- [ ] 現有方案不足說明

## 第 6-7 頁：創新構想
- [ ] 創新點明確（3-5 個）
- [ ] 每個創新點有量化效益
- [ ] 與競品比較表
- [ ] 技術門檻說明
- [ ] 創新可持續性

## 第 8-9 頁：可行性評估
- [ ] TRL 評估（現況與目標）
- [ ] 技術風險與對策
- [ ] 驗證計畫
- [ ] 時程規劃（甘特圖）

## 第 10-11 頁：市場分析
- [ ] TAM/SAM/SOM
- [ ] 目標客戶定義
- [ ] 客戶驗證證據
- [ ] 商業模式
- [ ] 競爭分析

## 第 12-13 頁：團隊組成
- [ ] 團隊架構圖
- [ ] 主持人介紹
- [ ] 核心成員介紹
- [ ] 能力矩陣
- [ ] 過往實績

## 第 14-15 頁：經費規劃
- [ ] 經費總表
- [ ] 人事費明細
- [ ] 設備使用費明細
- [ ] 比例符合規定
- [ ] 成本效益分析

## 整體檢查
- [ ] 頁數符合規定（12-15 頁）
- [ ] 字體大小適中（≥ 14pt）
- [ ] 有圖表輔助說明
- [ ] 邏輯清晰連貫
- [ ] 無錯別字
