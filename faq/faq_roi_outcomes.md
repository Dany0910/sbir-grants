# FAQ：預期成果與投資效益

## 投資效益比 (ROI/ROAS)

### Q1: 審查委員如何評估補助效益？

**A**: 審查委員會用「投資人」的角度評估您的計畫：

**核心問題**：
- 政府補助您 X 元，能創造多少產值？
- 投資報酬率 (ROAS) 是多少？
- 這筆錢值得投資嗎？

**評分標準**：
- ROAS < 2 倍：❌ 效益不足，不通過
- ROAS = 2-3 倍：⚠️ 勉強及格，競爭力弱
- ROAS = 3-5 倍：✅ 合格，有競爭力
- ROAS > 5 倍：⭐ 優秀，高機率通過

**實際案例**：
```
補助金額：150 萬
預期產值：450 萬（3 年累積）
ROAS = 450 / 150 = 3 倍
評價：✅ 及格，但建議提高到 750 萬（5 倍）
```

---

### Q2: 產值要寫多少才算合理？

**A**: 最低標準是補助金額的 **3 倍**，建議目標是 **5 倍**。

**產業差異**：

| 產業類型 | 最低 ROAS | 建議 ROAS | 原因 |
|---------|----------|----------|------|
| 製造業 | 3 倍 | 4-5 倍 | 資本密集，回收較慢 |
| 軟體/數位 | 5 倍 | 7-10 倍 | 邊際成本低，擴展快 |
| 服務創新 | 4 倍 | 5-6 倍 | 人力密集，穩定成長 |
| 硬體開發 | 3 倍 | 4 倍 | 開發成本高，市場風險大 |

**範例**：
```
軟體業，補助 100 萬
最低產值：500 萬（5 倍）
建議產值：700-1,000 萬（7-10 倍）
```

---

### Q3: 補助 100 萬，產值寫 100 萬會過嗎？

**A**: **絕對不會過。** 這是最常見的致命錯誤。

**審查委員的想法**：
> 「產值只有 100 萬？那代表：
> 1. 沒有補助，你也會做這件事（沒有補助必要性）
> 2. 或者市場太小，做了也沒用（沒有商業價值）
> 3. 政府的錢等於白花了（沒有投資效益）
> → 直接不通過」

**正確做法**：
```
補助 100 萬 → 產值至少 300 萬（3 倍）

計算範例：
第 1 年：100 萬（試銷）
第 2 年：150 萬（擴展）
第 3 年：200 萬（穩定）
合計：450 萬（4.5 倍）✅
```

---

### Q4: 如何計算「預期產值」？

**A**: 產值 = 直接產值 + 間接產值 + 外溢效益

#### 1. 直接產值（營收）

**Phase 1 計算**：
```
預期產值 = 計畫結束後 3 年內累積營收

範例：
補助金額：150 萬
計畫期程：12 個月

第 1 年（計畫中）：0 元
第 2 年（計畫後）：200 萬
第 3 年：300 萬
第 4 年：400 萬

3 年累積 = 200 + 300 + 400 = 900 萬
ROAS = 900 / 150 = 6 倍 ✅
```

**Phase 2 計算**：
```
預期產值 = 計畫結束後 3-5 年內累積營收
```

#### 2. 間接產值

可計入但需說明：
- **成本節省**：每年節省 50 萬 x 3 年 = 150 萬
- **效率提升**：產能提升 20%，增加產值 100 萬/年
- **品質改善**：良率提升，減少損失 30 萬/年

#### 3. 外溢效益（加分項）

不能作為主要產值，但可加分：
- 專利授權收入
- 技術輸出（顧問費）
- 帶動上下游產業
- 就業創造

**完整範例**：
```
直接產值（營收）：600 萬
間接產值（成本節省）：150 萬
外溢效益（專利授權）：50 萬
總產值 = 800 萬

ROAS = 800 / 150 = 5.3 倍 ✅
```

### Q5: 產值數字要怎麼有依據？不能亂寫嗎？

**A**: **絕對不能亂寫。** 每個數字都要有計算依據。

**錯誤寫法**：
```
❌ 「預計 3 年創造 1,000 萬營收」
（沒說怎麼來的，審查委員會質疑）
```

**正確寫法**：
```
✅ 「預計 3 年創造 1,000 萬營收，計算依據：

市場規模：5 億/年（根據 XXX 研究報告）
目標市佔率：2%（第 3 年）
預期營收 = 5 億 x 2% = 1,000 萬

客戶開發計畫：
- 第 1 年：5 家客戶，單價 20 萬 = 100 萬
- 第 2 年：15 家客戶，單價 20 萬 = 300 萬
- 第 3 年：30 家客戶，單價 20 萬 = 600 萬
合計：1,000 萬

佐證資料：
- 已有 3 家潛在客戶簽署合作意向書（附件）
- 市場調查問卷顯示 80% 願意購買（附件）
```

**計算依據來源**：
1. **市場規模**：產業報告、政府統計、研究機構
2. **市佔率**：參考競品、自身能力、客戶訪談
3. **單價**：市場行情、成本加成、客戶願付價格
4. **客戶數**：銷售計畫、通路能力、歷史數據

### Q6: 如果是全新產品，沒有歷史數據怎麼辦？

**A**: 用「類比法」和「客戶驗證」建立依據。

#### 方法 1：類比法

```
參考類似產品的市場表現：

「本產品類似於 XX 公司的 YY 產品，
該產品上市 3 年累積營收 500 萬。

考慮我們的優勢（價格便宜 30%、功能多 20%），
保守估計可達其 80% = 400 萬」
```

#### 方法 2：客戶驗證

```
已完成 20 家潛在客戶訪談：
- 15 家表示「非常有興趣」（75%）
- 願付價格：15-25 萬/年
- 預估轉換率：50%

第 1 年客戶數 = 20 x 75% x 50% = 7.5 家（保守取 5 家）
第 1 年營收 = 5 x 20 萬 = 100 萬
```

#### 方法 3：試銷驗證

```
已完成小規模試銷：
- 試銷對象：3 家客戶
- 試銷期間：3 個月
- 試銷營收：15 萬

推估全年營收 = 15 萬 x 4 = 60 萬（單一客戶）
第 1 年目標：10 家客戶 = 600 萬
```

### Q7: 產值寫太高會不會被質疑「吹牛」？

**A**: 關鍵不是「高或低」，而是「有沒有依據」。

**安全策略**：

1. **用「保守估計」降低風險**
```
市場規模：10 億/年
理論市佔率：5% = 5,000 萬
保守估計：取 20% = 1,000 萬 ✅

說明：「考慮市場競爭、客戶開發難度，
實際目標設定為理論值的 20%」
```

2. **分階段目標**
```
短期（1 年）：100 萬（保守）
中期（2-3 年）：300-500 萬（合理）
長期（3-5 年）：800-1,000 萬（樂觀）

「先求穩定，再求成長」
```

3. **提供佐證**
```
- 客戶意向書（3 家）
- 市場調查報告
- 試用回饋
- 專家推薦信
```

**記住**：
- 太保守 → 審查委員覺得「沒價值」
- 太樂觀 → 審查委員覺得「不可信」
- **合理樂觀 + 有依據 = 最佳策略**

### Q8: Phase 1 和 Phase 2 的產值計算有什麼不同？

**A**: 主要差異在「計畫期程」和「商業化程度」。

| 項目 | Phase 1 | Phase 2 |
|------|---------|---------|
| 計畫期程 | 6-12 個月 | 2 年 |
| 補助金額 | 最高 150 萬 | 最高 1,200 萬 |
| 產值計算期 | 計畫後 3 年 | 計畫後 3-5 年 |
| 最低 ROAS | 3 倍 | 2.5 倍 |
| 重點 | 技術可行性 | 商業化能力 |

**Phase 1 範例**：
```
補助：150 萬
計畫：12 個月（技術開發）
產值計算：計畫結束後 3 年

第 1 年（計畫後）：200 萬
第 2 年：300 萬
第 3 年：400 萬
合計：900 萬（ROAS = 6 倍）
```

**Phase 2 範例**：
```
補助：1,000 萬
計畫：2 年（產品開發 + 市場推廣）
產值計算：計畫結束後 3-5 年

第 1 年（計畫後）：500 萬
第 2 年：1,000 萬
第 3 年：1,500 萬
第 4 年：2,000 萬
第 5 年：2,500 萬
合計：7,500 萬（ROAS = 7.5 倍）
```

---

## 總結：產值撰寫黃金法則

1. **最低標準**：補助 1 元 → 產值 3 元
2. **建議目標**：補助 1 元 → 產值 5 元
3. **每個數字都要有依據**（市場規模 x 市佔率）
4. **用保守估計降低風險**（理論值 x 20-50%）
5. **提供佐證資料**（意向書、調查、試銷）
6. **合理樂觀，不誇大**

記住：**SBIR 是投資，不是慈善。證明高 ROI 是通過的關鍵。**
