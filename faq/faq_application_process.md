# SBIR 申請流程常見問題

## 申請時程

### Q1: SBIR 什麼時候開放申請？
**A:** 通常每年 2-3 次，依公告為準。建議定期查看 SBIR 官網。

### Q2: 從申請到核定需要多久？
**A:** 約 3-6 個月
- 書面審查：1-2 個月
- 簡報審查：1 個月
- 核定公告：1-2 個月

### Q3: 可以提前準備嗎？
**A:** 可以！建議提前 2-3 個月準備計畫書。

### Q4: 錯過這次還有機會嗎？
**A:** 有，每年有 2-3 次申請機會。

## 線上申請

### Q5: 如何註冊帳號？
**A:** 至 SBIR 官網註冊，需公司統編與負責人資料。

### Q6: 可以多人共同填寫嗎？
**A:** 可以，但建議指定一人為主要填寫者。

### Q7: 送出後可以修改嗎？
**A:** 截止前可修改，截止後無法修改。

### Q8: 需要上傳哪些文件？
**A:** 
- 計畫書 PDF
- 公司登記證明
- 無欠稅證明
- 其他佐證文件

## 審查流程

### Q9: Phase 1 需要簡報嗎？
**A:** 需要，通過書面審查後需簡報（10-15 分鐘）

### Q10: Phase 2 需要簡報嗎？
**A:** 視情況，部分案件需要

### Q11: 審查委員會問什麼？
**A:** 
- 創新性：與競品差異
- 可行性：技術風險、團隊能力
- 市場性：市場規模、客戶驗證
- 經費：經費合理性

### Q12: 可以重新申請嗎？
**A:** 可以，但需調整計畫內容

### Q13: 通過率大概多少？
**A:** 
- Phase 1：約 40-50%
- Phase 2：約 25-35%
