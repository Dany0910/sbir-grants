# 預期成果與效益撰寫方法論

## 核心觀念：SBIR 是政府的投資決策

### ⚠️ 關鍵認知

**SBIR 補助不是慈善，而是投資。**

審查委員評估您的計畫時，會用「投資人」的角度思考：
- 政府補助您 100 萬，能創造多少產值？
- 這筆投資的報酬率 (ROI/ROAS) 是多少？
- 如果產值只有 100 萬，等於「政府出錢讓您做本來就會做的事」→ **直接不通過**

### 💰 投資效益比 (ROAS) 基準

| 補助金額 | 最低產值 | 建議產值 | ROAS |
|---------|---------|---------|------|
| 100 萬 | 300 萬 | 500 萬 | 3-5 倍 |
| 150 萬 | 450 萬 | 750 萬 | 3-5 倍 |
| 500 萬 | 1,500 萬 | 2,500 萬 | 3-5 倍 |

**產業差異**：
- 製造業：3-5 倍
- 軟體/數位服務：5-10 倍
- 服務創新：4-6 倍
- 硬體開發：3-4 倍

## 產值計算公式

### 1. 直接產值（營收）

**Phase 1 (6-12 個月計畫)**：
```
預期產值 = 計畫結束後 3 年內累積營收
```

**Phase 2 (2 年計畫)**：
```
預期產值 = 計畫結束後 3-5 年內累積營收
```

**計算範例**：
```
補助金額：150 萬
計畫期程：Phase 1 (12 個月)

第 1 年（計畫執行中）：0 元（開發階段）
第 2 年（計畫結束）：200 萬（初期銷售）
第 3 年：300 萬（市場擴展）
第 4 年：400 萬（穩定成長）

3 年累積產值 = 200 + 300 + 400 = 900 萬
ROAS = 900 / 150 = 6 倍 ✅ 合格
```

### 2. 間接產值

除了營收，還可計入：
- **成本節省**：導入新技術後，每年節省的成本
- **效率提升**：生產效率提升帶來的產能增加
- **品質改善**：良率提升減少的損失

**範例**：
```
直接產值（營收）：600 萬
間接產值（成本節省）：每年 50 萬 x 3 年 = 150 萬
總產值 = 600 + 150 = 750 萬
```

### 3. 外溢效益

**可加分項目**（但不能作為主要產值）：
- 專利授權收入
- 技術輸出（顧問、教育訓練）
- 帶動上下游產業發展
- 就業創造（每增加 1 個工作機會約值 50-80 萬/年）

## 常見錯誤與修正

### ❌ 錯誤 1：產值過於保守

**錯誤寫法**：
> 「預計 3 年內創造 150 萬營收」（補助 150 萬）

**問題**：ROAS = 1，審查委員會認為「沒有補助效益」。

**正確寫法**：
> 「預計 3 年內創造 750 萬營收，其中：
> - 第 1 年：100 萬（試銷階段）
> - 第 2 年：250 萬（市場擴展）
> - 第 3 年：400 萬（穩定成長）
> ROAS = 5 倍」

### ❌ 錯誤 2：只寫技術成果，不寫商業效益

**錯誤寫法**：
> 「完成 3 項專利申請、發表 2 篇論文」

**問題**：審查委員想知道「這些專利能賣多少錢？」

**正確寫法**：
> 「完成 3 項專利申請，預計：
> - 專利 A：授權給 5 家廠商，每家年費 20 萬 = 100 萬/年
> - 專利 B、C：自行應用於產品，帶動產品銷售 500 萬/年
> 3 年累積效益 = (100 + 500) x 3 = 1,800 萬」

### ❌ 錯誤 3：數字沒有依據

**錯誤寫法**：
> 「預計創造 1,000 萬營收」（沒有說明怎麼來的）

**正確寫法**：
> 「預計創造 1,000 萬營收，計算依據：
> - 目標市場規模：5 億/年
> - 目標市佔率：2%（第 3 年）
> - 預期營收 = 5 億 x 2% = 1,000 萬
> - 已有 3 家潛在客戶簽署合作意向書（附件）」

## 撰寫架構

### 第一部分：量化成果（60%）

#### 1. 營收預估
```markdown
## 預期營收

| 年度 | 營收（萬） | 說明 |
|------|-----------|------|
| 第 1 年 | 100 | 試銷階段，5 家客戶 |
| 第 2 年 | 300 | 擴展至 15 家客戶 |
| 第 3 年 | 500 | 穩定成長，20 家客戶 |
| **合計** | **900** | **ROAS = 6 倍** |

**計算依據**：
- 單一客戶年均消費：20 萬
- 客戶獲取成本：5 萬/家
- 客戶留存率：80%
```

#### 2. 市場佔有率
```markdown
## 市場佔有率提升

- 補助前：0%（新產品）
- 第 1 年：0.5%
- 第 2 年：1.2%
- 第 3 年：2.0%

**市場規模**：500 億/年（根據 XXX 研究報告）
**第 3 年營收** = 500 億 x 2% = 10 億（保守估計取 5% = 5,000 萬）
```

#### 3. 成本效益
```markdown
## 成本節省效益

導入新技術後：
- 生產時間縮短 30%：每年節省人力成本 80 萬
- 良率提升 15%：每年減少損失 50 萬
- 能源效率提升 20%：每年節省 30 萬

**3 年累積節省** = (80 + 50 + 30) x 3 = 480 萬
```

### 第二部分：質化成果（20%）

- 技術突破（專利、論文）
- 人才培育（培訓人數、證照）
- 產業影響（標準制定、示範效應）

### 第三部分：社會效益（20%）

- 就業創造
- 環境保護
- 區域發展

## 不同階段的撰寫重點

### Phase 1（創新研發）

**重點**：證明「技術可行 + 市場潛力」

```markdown
## 預期成果

### 技術成果
- 完成技術驗證（TRL 4 → TRL 6）
- 申請 2 項專利

### 商業成果
- 取得 5 家客戶試用承諾
- 預估 3 年累積營收：800 萬
- ROAS：5.3 倍

### 計算依據
（詳細說明）
```

### Phase 2（產品開發）

**重點**：證明「商業化能力」

```markdown
## 預期成果

### 市場成果
- 第 1 年：銷售 50 套，營收 500 萬
- 第 2 年：銷售 100 套，營收 1,000 萬
- 第 3 年：銷售 150 套，營收 1,500 萬
- **3 年累積：3,000 萬（ROAS = 2.5 倍）**

### 市場擴展
- 建立 3 個區域經銷商
- 進入 2 個海外市場

### 產業影響
- 帶動上游供應鏈 5 家廠商
- 創造 15 個就業機會
```

## 檢核清單

撰寫完成後，請自我檢查：

- [ ] **ROAS ≥ 3 倍**（最低標準）
- [ ] **營收預估有明確計算依據**（市場規模 x 市佔率）
- [ ] **數字合理樂觀**（不過度誇大，但也不過於保守）
- [ ] **有客戶驗證**（意向書、試用承諾、問卷調查）
- [ ] **分年預估**（不要只寫總數）
- [ ] **包含間接效益**（成本節省、效率提升）
- [ ] **連結到市場分析**（與前面章節的市場規模一致）

## 實戰技巧

### 技巧 1：用「保守估計」降低風險

```markdown
市場規模：10 億/年
目標市佔率：5%（第 3 年）
理論營收：5,000 萬

**保守估計**：考慮市場競爭、客戶開發難度，
實際營收目標設定為理論值的 20% = 1,000 萬

ROAS = 1,000 / 150 = 6.7 倍 ✅
```

### 技巧 2：用「客戶承諾」增加可信度

```markdown
已取得 3 家潛在客戶簽署合作意向書：
- A 公司：預計年採購 100 萬
- B 公司：預計年採購 80 萬
- C 公司：預計年採購 60 萬

保守估計達成率 50%，第 1 年營收 = 120 萬
```

### 技巧 3：用「階段目標」展現務實

```markdown
## 分階段目標

### 短期（1 年內）
- 完成產品開發
- 取得 5 家試用客戶
- 營收：100 萬

### 中期（2-3 年）
- 擴展至 20 家客戶
- 建立經銷通路
- 營收：300-500 萬/年

### 長期（3-5 年）
- 市佔率達 2%
- 進入海外市場
- 營收：800-1,000 萬/年
```

## 總結

記住三個關鍵：
1. **SBIR 是投資，不是補助** → 要證明高 ROI
2. **數字要有依據** → 市場規模 x 市佔率
3. **合理樂觀** → 不誇大，但也不過度保守

**最低標準**：補助 1 元 → 產值 3 元
**建議目標**：補助 1 元 → 產值 5 元
