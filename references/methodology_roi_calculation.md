# 投資效益比 (ROI/ROAS) 計算方法論

## 為什麼 SBIR 重視 ROI？

### 政府的角度

SBIR 補助來自納稅人的錢，政府必須確保每一分錢都花得有價值：

**政府的三個核心問題**：
1. 這筆投資能創造多少經濟價值？
2. 投資報酬率 (ROAS) 是多少？
3. 為什麼要補助你，而不是其他公司？

**審查委員的評估邏輯**：
```
補助 100 萬給 A 公司 → 3 年創造 500 萬產值 → ROAS = 5 倍 ✅
補助 100 萬給 B 公司 → 3 年創造 150 萬產值 → ROAS = 1.5 倍 ❌

結論：A 公司通過，B 公司不通過
```

### 企業的角度

**對您的好處**：
- 清楚的產值目標 → 更明確的商業計畫
- 高 ROI 預估 → 更容易獲得補助
- 量化的成果 → 更容易向投資人募資

## 產值計算公式

### 公式 1：直接產值（營收）

這是最重要的產值來源。

**Phase 1 (6-12 個月計畫)**：
```
預期產值 = 計畫結束後 3 年內累積營收

範例：
補助金額：150 萬
計畫期程：12 個月

第 1 年（計畫執行中）：0 元（開發階段，無營收）
第 2 年（計畫結束後第 1 年）：200 萬
第 3 年（計畫結束後第 2 年）：300 萬
第 4 年（計畫結束後第 3 年）：400 萬

3 年累積產值 = 200 + 300 + 400 = 900 萬
ROAS = 900 / 150 = 6 倍 ✅
```

**Phase 2 (2 年計畫)**：
```
預期產值 = 計畫結束後 3-5 年內累積營收

範例：
補助金額：1,000 萬
計畫期程：2 年

第 1-2 年（計畫執行中）：0 元
第 3 年（計畫結束後第 1 年）：500 萬
第 4 年：1,000 萬
第 5 年：1,500 萬
第 6 年：2,000 萬
第 7 年：2,500 萬

5 年累積產值 = 500 + 1,000 + 1,500 + 2,000 + 2,500 = 7,500 萬
ROAS = 7,500 / 1,000 = 7.5 倍 ✅
```

**計算依據**：
```
營收 = 客戶數量 x 單一客戶年均消費

範例：
第 1 年：5 家客戶 x 40 萬/家 = 200 萬
第 2 年：15 家客戶 x 20 萬/家 = 300 萬
第 3 年：30 家客戶 x 13.3 萬/家 = 400 萬
```

### 公式 2：間接產值

除了營收，還可計入：

#### 2.1 成本節省
```
導入新技術後，每年節省的成本

範例：
- 生產時間縮短 30% → 每年節省人力成本 80 萬
- 良率提升 15% → 每年減少損失 50 萬
- 能源效率提升 20% → 每年節省 30 萬

3 年累積節省 = (80 + 50 + 30) x 3 = 480 萬
```

#### 2.2 效率提升
```
產能提升帶來的額外產值

範例：
- 生產效率提升 20%
- 原本年產值 1,000 萬
- 提升後年產值 1,200 萬
- 額外產值 = 200 萬/年

3 年累積 = 200 x 3 = 600 萬
```

#### 2.3 品質改善
```
良率提升減少的損失

範例：
- 良率從 85% 提升到 95%
- 每年生產 1,000 件，單價 1 萬
- 原本損失：150 件 x 1 萬 = 150 萬
- 提升後損失：50 件 x 1 萬 = 50 萬
- 每年減少損失 = 100 萬

3 年累積 = 100 x 3 = 300 萬
```

### 公式 3：外溢效益

這些是「加分項」，不能作為主要產值，但可以提升評分：

#### 3.1 專利授權
```
專利授權收入

範例：
- 申請 3 項專利
- 專利 A：授權給 5 家廠商，每家年費 20 萬 = 100 萬/年
- 專利 B、C：自行應用

3 年累積 = 100 x 3 = 300 萬
```

#### 3.2 技術輸出
```
顧問、教育訓練收入

範例：
- 每年接 5 個顧問案，每案 30 萬 = 150 萬/年
- 3 年累積 = 450 萬
```

#### 3.3 就業創造
```
每增加 1 個工作機會約值 50-80 萬/年

範例：
- 計畫執行後增加 10 個工作機會
- 10 x 60 萬 x 3 年 = 1,800 萬
```

### 完整計算範例

```
補助金額：150 萬

直接產值（營收）：
- 第 1 年：200 萬
- 第 2 年：300 萬
- 第 3 年：400 萬
小計：900 萬

間接產值：
- 成本節省：480 萬
- 效率提升：600 萬
小計：1,080 萬

外溢效益：
- 專利授權：300 萬
- 就業創造：1,800 萬
小計：2,100 萬

總產值 = 900 + 1,080 + 2,100 = 4,080 萬
ROAS = 4,080 / 150 = 27.2 倍 ⭐⭐⭐

（但實務上，主要看直接產值 900 萬 = 6 倍，已經很優秀）
```

## 不同產業的 ROAS 基準

### 製造業：3-5 倍

**特性**：
- 資本密集
- 回收期較長
- 需要設備投資

**合理範圍**：
- 最低：3 倍
- 建議：4-5 倍
- 優秀：6 倍以上

**範例**：
```
補助 150 萬（機械設備開發）
3 年累積產值：600-750 萬
ROAS：4-5 倍 ✅
```

### 軟體/數位服務：5-10 倍

**特性**：
- 邊際成本低
- 擴展速度快
- 可快速複製

**合理範圍**：
- 最低：5 倍
- 建議：7-10 倍
- 優秀：12 倍以上

**範例**：
```
補助 100 萬（SaaS 平台開發）
3 年累積產值：700-1,000 萬
ROAS：7-10 倍 ✅
```

### 服務創新：4-6 倍

**特性**：
- 人力密集
- 穩定成長
- 可複製性中等

**合理範圍**：
- 最低：4 倍
- 建議：5-6 倍
- 優秀：8 倍以上

**範例**：
```
補助 120 萬（服務模式創新）
3 年累積產值：600-720 萬
ROAS：5-6 倍 ✅
```

### 硬體開發：3-4 倍

**特性**：
- 開發成本高
- 市場風險大
- 認證時間長

**合理範圍**：
- 最低：3 倍
- 建議：4 倍
- 優秀：5 倍以上

**範例**：
```
補助 150 萬（IoT 裝置開發）
3 年累積產值：450-600 萬
ROAS：3-4 倍 ✅
```

## 實際案例演算

### 案例 1：製造業（機械零件）

**背景**：
- 公司：傳統機械零件製造
- 計畫：開發智慧化生產系統
- 補助金額：150 萬
- 計畫期程：Phase 1, 12 個月

**產值計算**：

**直接產值（營收）**：
```
市場規模：台灣機械零件市場 500 億/年
目標客群：中小型製造業（200 家）
目標市佔率：
- 第 1 年：2 家（1%）
- 第 2 年：6 家（3%）
- 第 3 年：10 家（5%）

單一客戶年均消費：40 萬（設備 + 維護）

營收預估：
- 第 1 年：2 x 40 = 80 萬
- 第 2 年：6 x 40 = 240 萬
- 第 3 年：10 x 40 = 400 萬
合計：720 萬
```

**間接產值**：
```
自身工廠導入後：
- 生產效率提升 25% → 每年增加產值 100 萬
- 良率提升 10% → 每年減少損失 50 萬
3 年累積：(100 + 50) x 3 = 450 萬
```

**總產值**：
```
直接產值：720 萬
間接產值：450 萬
總計：1,170 萬

ROAS = 1,170 / 150 = 7.8 倍 ⭐
```

**計算依據**：
- 市場規模：根據工研院產業報告
- 客戶數量：已有 3 家簽署合作意向書
- 單價：參考市場行情 + 成本加成
- 效率提升：試驗場域驗證數據

---

### 案例 2：軟體業（SaaS）

**背景**：
- 公司：軟體新創
- 計畫：開發 AI 客服 SaaS 平台
- 補助金額：100 萬
- 計畫期程：Phase 1, 12 個月

**產值計算**：

**直接產值（營收）**：
```
市場規模：台灣中小企業 150 萬家
目標客群：有客服需求的中小企業（30 萬家）
目標市佔率：
- 第 1 年：50 家（0.017%）
- 第 2 年：200 家（0.067%）
- 第 3 年：500 家（0.167%）

訂閱制：每家每月 5,000 元 = 6 萬/年

營收預估：
- 第 1 年：50 x 6 = 300 萬
- 第 2 年：200 x 6 = 1,200 萬
- 第 3 年：500 x 6 = 3,000 萬
合計：4,500 萬
```

**總產值**：
```
直接產值：4,500 萬
ROAS = 4,500 / 100 = 45 倍 ⭐⭐⭐
```

**計算依據**：
- 市場規模：經濟部中小企業統計
- 客戶數量：已完成 20 家試用，轉換率 50%
- 訂閱價格：市場調查，80% 願意支付 5,000/月
- 成長曲線：參考類似 SaaS 產品（Zendesk, Intercom）

## 撰寫技巧

### 技巧 1：用「保守估計」降低風險

```
市場規模：10 億/年
理論市佔率：5%（第 3 年）
理論營收：5,000 萬

保守估計：考慮市場競爭、客戶開發難度
實際目標：理論值的 20% = 1,000 萬

ROAS = 1,000 / 150 = 6.7 倍 ✅

說明：「雖然理論市佔率可達 5%，但考慮實際執行難度，
我們保守估計為 1%，對應營收 1,000 萬」
```

### 技巧 2：分階段目標

```
短期（1 年內）：
- 完成產品開發
- 取得 5 家試用客戶
- 營收：100 萬

中期（2-3 年）：
- 擴展至 20 家客戶
- 建立經銷通路
- 營收：300-500 萬/年

長期（3-5 年）：
- 市佔率達 2%
- 進入海外市場
- 營收：800-1,000 萬/年
```

### 技巧 3：提供多種情境

```
保守情境（機率 70%）：
- 客戶數：10 家
- 營收：400 萬
- ROAS：2.7 倍

基準情境（機率 20%）：
- 客戶數：20 家
- 營收：800 萬
- ROAS：5.3 倍

樂觀情境（機率 10%）：
- 客戶數：30 家
- 營收：1,200 萬
- ROAS：8 倍

「我們以基準情境作為目標，即使在保守情境下，
ROAS 仍達 2.7 倍，具備一定安全邊際」
```

## 檢核清單

計算完成後，請確認：

- [ ] **ROAS ≥ 產業基準**（製造 3-5 倍、軟體 5-10 倍）
- [ ] **每個數字都有依據**（市場規模、市佔率、單價）
- [ ] **計算邏輯清楚**（營收 = 客戶數 x 單價）
- [ ] **有佐證資料**（意向書、試用數據、市場報告）
- [ ] **分年預估合理**（成長曲線符合產業特性）
- [ ] **用保守估計**（理論值 x 20-50%）
- [ ] **提供計算依據**（不是憑空想像）

---

## 總結

**三個關鍵公式**：
1. 直接產值 = 客戶數 x 單價 x 年數
2. 間接產值 = 成本節省 + 效率提升
3. ROAS = 總產值 / 補助金額

**三個基本原則**：
1. 最低標準：ROAS ≥ 3 倍
2. 每個數字都要有依據
3. 合理樂觀，不誇大

**記住**：SBIR 是投資，不是慈善。證明高 ROI 是通過的關鍵。
