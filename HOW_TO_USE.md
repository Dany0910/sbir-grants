# 🎯 如何使用 SBIR Skill

> 安裝完成後，Claude 會自動使用知識庫來回答您的問題！

---

## ✨ 自動化魔法

**好消息**：您不需要記住任何指令或檔案路徑！

Claude 會自動：
1. 🔍 搜尋相關的知識庫文件
2. 📖 讀取文件內容
3. 💡 根據內容回答您的問題

---

## 🗣️ 直接問問題就好

### 範例 1：我要寫創新構想

**您問**：
```
我要寫創新構想這個章節，該怎麼寫？
```

**Claude 會自動**：
1. 搜尋知識庫找到 `methodology_innovation.md`
2. 讀取方法論內容
3. 根據方法論指導您撰寫

---

### 範例 2：我想知道資格

**您問**：
```
我的公司實收資本額 5000 萬，員工 50 人，可以申請 SBIR 嗎？
```

**Claude 會自動**：
1. 搜尋 FAQ 中的資格相關問題
2. 檢查資格條件
3. 告訴您是否符合

---

### 範例 3：我要查市場數據

**您問**：
```
請幫我找機械產業的市場數據
```

**Claude 會自動**：
1. 使用 MCP Server 查詢經濟部數據
2. 使用 search_web 查詢 IEK、MIC
3. 整合多個來源的數據

---

## 📚 可以問的問題類型

### 💡 關於撰寫

- 「如何寫問題陳述？」
- 「市場分析要包含哪些內容？」
- 「經費要怎麼編列？」
- 「團隊組成要寫什麼？」

### ❓ 關於資格

- 「我能申請 SBIR 嗎？」
- 「Phase 1 和 Phase 2 有什麼差別？」
- 「外資持股超過 50% 可以申請嗎？」

### 📊 關於數據

- 「請幫我找 [產業] 的市場數據」
- 「TAM/SAM/SOM 怎麼計算？」
- 「哪裡可以找到產業統計？」

### ✅ 關於檢查

- 「幫我檢查計畫書是否完整」
- 「送件前要注意什麼？」
- 「經費編列有什麼限制？」

## 🎯 完整工作流程

### 步驟 1：確認資格（10 分鐘）

**問 Claude**：
```
幫我確認 SBIR 申請資格，我的公司...
```

---

### 步驟 2：驗證構想（1 小時）

**問 Claude**：
```
我想做 [您的構想]，幫我用 1 小時構想驗證的方法評估一下
```

---

### 步驟 3：開始撰寫（1-4 週）

**逐章撰寫**，每個章節都問 Claude：

```
我要寫 [章節名稱]，請提供方法論和範本
```

**範例**：
- 「我要寫問題陳述」
- 「我要寫創新構想」
- 「我要寫市場分析」
- 「我要寫可行性評估」
- 「我要寫團隊組成」
- 「我要寫經費規劃」

---

### 步驟 4：品質檢查

**問 Claude**：
```
幫我檢查 Phase 1 計畫書是否完整
```

---

### 步驟 5：送件

**問 Claude**：
```
送件前要準備哪些文件？
```

---

## 🆘 遇到問題？

### Claude 沒有使用知識庫？

**可能原因**：
1. MCP Server 沒有啟動 → 重啟 Claude Desktop
2. 問題太模糊 → 問得更具體一點

**解決方法**：
明確要求 Claude 使用知識庫：
```
請搜尋 SBIR 知識庫，找到關於 [主題] 的文件
```
