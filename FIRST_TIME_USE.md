# 📦 第一次使用指南

> 安裝完成後，該如何開始使用 SBIR Skill？

---

## 🎯 第一次使用：問 Claude 這些問題

安裝完成後，打開 Claude Desktop，試試看問這些問題：

### 1️⃣ 確認資格

```
我的公司實收資本額 5000 萬，員工 50 人，可以申請 SBIR 嗎？
```

**Claude 會**：
- 檢查您的資格
- 告訴您是否符合條件
- 建議下一步

---

### 2️⃣ 驗證構想

```
我想做一個 [您的構想]，這個適合申請 SBIR 嗎？
```

**範例**：
```
我想做一個 AI 客服系統給中小企業用，這個適合申請 SBIR 嗎？
```

**Claude 會**：
- 評估創新性
- 建議使用構想驗證指南
- 提供改進方向

---

### 3️⃣ 開始撰寫

```
我要寫 SBIR Phase 1 計畫書，該從哪裡開始？
```

**Claude 會**：
- 提供完整的撰寫流程
- 推薦使用範本和方法論
- 引導您一步步完成

---

### 4️⃣ 查詢市場數據

```
請幫我找機械產業的市場數據，我要寫市場分析
```

**Claude 會**：
- 使用 MCP Server 查詢
- 使用 search_web 找資料
- 提供數據來源和計算方法

---

## 📚 完整學習路徑

**想要系統性地學習？** 照這個順序：

1. **了解 SBIR**（20 分鐘）  
   👉 閱讀 [`GETTING_STARTED.md`](GETTING_STARTED.md) 的 Step 0

2. **10 分鐘資格檢查**（10 分鐘）  
   👉 使用 [`quick_start/10min_eligibility_check.md`](quick_start/10min_eligibility_check.md)

3. **1 小時構想驗證**（60 分鐘）  
   👉 使用 [`quick_start/1hour_idea_validation.md`](quick_start/1hour_idea_validation.md)

4. **開始撰寫計畫書**（1-4 週）  
   👉 參考 [`GETTING_STARTED.md`](GETTING_STARTED.md) 的 Step 3
