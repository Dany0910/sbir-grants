# SBIR 計畫書範本

## 概述

本目錄包含 SBIR 計畫書相關範本與官方下載連結。

---

## 📝 可填寫範本（Markdown）

本專案提供可直接使用的 Markdown 範本，適合與 Claude 或其他 AI 工具協作撰寫：

| 範本 | 適用階段 | 說明 |
|------|----------|------|
| [phase1_proposal.md](phase1_proposal.md) | Phase 1 先期研究 | 簡報格式，約 12-15 頁 |
| [phase2_proposal.md](phase2_proposal.md) | Phase 2 研究開發 | 完整計畫書，約 30-40 頁 |
| [budget_template.md](budget_template.md) | Phase 1/2 通用 | 經費編列計算表 |

### 使用方式

1. 複製對應的範本檔案
2. 填寫所有 `[待填寫]` 欄位
3. 參考 `<!-- 填寫指引 -->` 中的建議
4. 完成後移除所有註解
5. 轉換為 Word/PDF 或簡報格式提交

### 範本特色

- ✅ **結構完整**：符合官方最新要求（115年度/2026年）
- ✅ **填寫指引**：每個章節都有撰寫建議
- ✅ **檢核清單**：內建自我檢查表
- ✅ **表格格式**：方便數據整理與呈現

## 範本使用建議

### Phase 1 簡報範本

Phase 1 通常以簡報審查為主，建議結構：

```
1. 封面 (1頁)
2. 公司簡介 (2-3頁)
3. 問題陳述 (2-3頁)
4. 創新構想 (3-4頁)
5. 可行性評估規劃 (2-3頁)
6. 市場初步分析 (1-2頁)
7. 團隊介紹 (1頁)
8. 時程與經費 (1頁)
9. 結語 (1頁)
```

總頁數：約 15-20 頁

### Phase 2 計畫書範本

Phase 2 需完整計畫書，建議結構：

```
1. 封面
2. 摘要 (1頁)
3. 目錄
4. 公司概況 (2-3頁)
5. 計畫背景與目標 (5-8頁)
6. 研發方法與執行策略 (8-12頁)
7. 市場分析與商業化規劃 (5-8頁)
8. 團隊能力 (2-3頁)
9. 經費編列 (2-3頁)
10. 預期成果與效益 (2-3頁)
11. 附件
```

總頁數：約 30-40 頁
