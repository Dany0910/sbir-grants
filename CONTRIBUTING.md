# 貢獻指南

感謝您對 SBIR Skill 專案的興趣！

## 如何貢獻

### 1. 回報問題

如發現錯誤或有改進建議，請：
- 檢查是否已有類似 issue
- 提供詳細描述與範例
- 註明相關檔案路徑

### 2. 提交內容

歡迎貢獻：
- 新的方法論
- 實戰案例（需為合成範例）
- FAQ 問題與解答
- 錯誤修正

### 3. 內容規範

#### 方法論檔案
- 必須包含免責聲明
- 至少 2-3 個範例
- 包含檢核清單
- 有參考資源連結

#### 案例研究
- 必須為合成範例
- 包含評分與審查問答
- 說明成功/失敗關鍵

#### FAQ
- 問題明確
- 答案具體
- 包含範例

### 4. Markdown 格式

- 使用表格呈現比較數據
- 使用 emoji 增加可讀性
- 使用 blockquote 標註重要提示
- 使用 code block 標註範例

### 5. 提交流程

1. Fork 專案
2. 建立分支
3. 提交變更
4. 發送 Pull Request
